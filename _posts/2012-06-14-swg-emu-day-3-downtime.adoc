Tonight things didn't go as planned. The server was having stability issues. So, we didn't get to really do too much. We did manage to give some meatlumps a bad day. I also installed the emu on another computer. Now we have the emu installed on both desktops and laptops. If I could only get it installed on the iPad I would be in heaven.

I really wish I had more to write, but bedtime is approaching fast. One important note for the emu today is server load. They managed to get over 1900 players on at the same time. This could also be part of the stability issues.

There are debates on how many were on a live SWG server. The consensus seems to be 2000 -- 3000 per server. This means the Emu has managed to handle the same load as a standard server.
